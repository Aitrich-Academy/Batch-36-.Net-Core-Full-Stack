More about Article tag

The <article> tag specifies independent, self-contained content.
An article should make sense on its own and it should be possible
to distribute it independently from the rest of the site.

Note: The ⁢article> element does not render as anything special in a browser.
However, you can use CSS to style the ⁢article> element
